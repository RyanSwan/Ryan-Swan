My Blogs

Python Exam

Date: 1st November 2023

Had my first exam on the semester and I have to say it was an incredible experience. Learning a coding language is something that I have wanted to do for a long time now and the first six weeks of this semester has been a grind that I have taken in my stride. The exam consisted of the basics of python (input function was the first question), and working with libraries such as numpy and pandas. I do like numpy, however I think pandas is far more gripping and challenging. Manipulating dataframes is something I really find myself interested in. I think I did pretty well on the exam, although there was a functions question that I struggled with. Overall, I'm really happy with how the data analytics course is going so far!

Machine Learning Exam

Date: 29th November 2023

Today marked the second exam of the semester, this time it was for the MT413 module: 'Data Mining and Predictive Analytics'. This exam was composed of the various elements we have studied for the past two months, such as data cleaning, feature engineering, and the implementation of machine learning algorithms. At first, learning about the theory of these topics was quite intimidating to me, as they seemed to be very high level. However, I could not be happier with how that exam went! I'm very happy with how I cleaned the data, I knew how to implement dummy variables and an encoding dictionary, and my models had a high accuracy measure. Another step in the right direction!
