#Standardization (Z-score): The formula (value - mean) / standard deviation is applied to each value in avgMtlyHrs.
#This process standardizes the data, transforming it into z-scores. Z-scores express how many standard deviations a data point is from the mean.
#By standardizing the data, you are transforming it into a common scale, making it easier to compare the relative magnitudes of different features.
#This process is often beneficial for certain algorithms and analyses, especially those that rely on distance metrics or where the scale of features might affect model performance.
